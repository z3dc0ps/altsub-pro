## Altsub-pro
Tool For Finding Available Subdomains
## Installation
```bash
apt-get install golang
go get github.com/gookit/color
git clone https://github.com/z3dc0ps/altsub-pro
```
## Usage ( Recommended )
```bash
go run main.go
```
