## Altsub-pro
Tool For Scanning And Finding Available Subdomains
## Installation
```bash
go get github.com/gookit/color
git clone https://github.com/z3dc0ps/altsub-pro
```
## Usage ( Recommended )
```bash
go run main.go
```
